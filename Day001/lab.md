# Day001 Lab

Topic: Fixed Broadband Overview

Status: Template

## Lab Goal

Observe the local network from the learner's Mac and connect the observation to fixed broadband concepts.

## Environment

- Mac
- Terminal
- Wireshark, if packet observation is needed

## Steps

1. Identify the local IP address.
2. Identify the default gateway.
3. Identify the DNS server.
4. Observe how the Mac reaches the gateway.
5. Record the result in the lab notes.

## Expected Result

The learner can explain, at a basic level, how their device connects to the local network and why the gateway matters.
