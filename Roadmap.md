# Roadmap

This file maintains the 120-day learning route only.

Course content belongs in each `DayXXX` folder, not here.

## Stage 1: Foundation and Big Picture

Goal: build a broad mental map of fixed broadband, networking basics, and operator access networks.

Planned days: Day001 - Day030

Focus:

- Network fundamentals
- Fixed broadband architecture overview
- Access, aggregation, and core concepts
- Basic Ethernet, IP, ARP, VLAN, routing, and DNS concepts
- Wireshark-based observation

## Stage 2: Protocols and Operator Practice

Goal: deepen understanding of protocols and how operators use them in real networks.

Planned days: Day031 - Day070

Focus:

- VLAN and QinQ
- PPPoE and IPoE
- DHCP
- OSPF and IS-IS basics
- BNG/BRAS concepts
- Huawei VRP-style configuration thinking
- `display`-oriented troubleshooting

## Stage 3: Architecture, Troubleshooting, and Automation

Goal: connect theory, engineering practice, troubleshooting, and automation.

Planned days: Day071 - Day120

Focus:

- PON and FTTH architecture
- QoS and multicast basics
- Common fixed broadband failures
- Troubleshooting methodology
- Container-based labs with Linux, FRRouting, and containerlab
- Automation thinking for operator networks

## Notes

The roadmap should remain lightweight. Adjustments should preserve the project goal: helping the learner steadily master fixed broadband without increasing maintenance cost.
